Boilerplate is basically an HTML template.

It comes from the old days when the printing press used iron plates as their templetes.

Use "em" tag instead of "i" tag whenever you can.

This tells the browser that the text needs to be stressed and also conveys more information then the "i" tag

The same can be said about the "strong" tag.

The alt is useful when trying to get your site to show up when in searches.

So make it descriptive and add what is important.
